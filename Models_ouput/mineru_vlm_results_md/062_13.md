# A. Robust Principal Component Analysis (RPCA)

RPCA is a powerful mathematical tool that has been used in many scientific domains in order to decompose an observed measurement  $\mathbf{E}$  into a low- rank matrix  $\mathbf{L}$ , representing the ideal data unaffected by any kind of noise, and a sparse matrix  $\mathbf{S}$ , representing the noisy data. Decomposition is performed by solving the following equation:

$$
\underset {\mathbf{L},\mathbf{S}}{\arg \min}\| \mathbf{L}\|_{*} + \lambda \| \mathbf{S}\|_{1},\quad \mathrm{s.t.}\mathbf{L} + \mathbf{S} = \mathbf{E}, \tag{9}
$$

where  $\| \mathbf{L}\|_{*}$  is the nuclear norm of a matrix  $\mathbf{L}$  (i.e.,  $\sum_{i}\sigma_{i}(\mathbf{L})$  is the sum of the singular values of  $\mathbf{L}$ ).

A lot of works have been proposed all of these years, presenting excellent results. However, despite the effectiveness that some works [72], [73] have presented in the past, the execution times of the proposed algorithms need improvement. This convex problem can be solved using a very fast approach, as described in [74], according to:

$$
\begin{array}{rl} & {\underset {\mathbf{L},\mathbf{S}}{\arg \min}\frac{1}{2}\| \mathbf{L} + \mathbf{S} - \mathbf{E}\| _F + \lambda \| \mathbf{S}\| _1\quad \mathrm{s.t.}\mathrm{rank}(\mathbf{L}) = K}\\ & {\mathbf{L}^{(t + 1)} = \underset {\mathbf{L}}{\arg \min}\| \mathbf{L} + \mathbf{S}^{(t)} - \mathbf{E}\| _F\quad \mathrm{s.t.}\mathrm{rank}(\mathbf{L}) = K} \end{array} \tag{10}
$$

$$
\mathbf{S}^{(t + 1)} = \underset {\mathbf{S}}{\arg \min}\| \mathbf{L}^{(t + 1)} + \mathbf{S} - \mathbf{E}\| _F + \lambda \| \mathbf{S}\| _1 \tag{12}
$$

In each  $(t)$  iteration, the Eq. (11) is updated with  $\mathrm{rank} = K$ . If  $\frac{u_K}{\sum_{i = 1}^{K}u_i} >\epsilon$ , where  $u$  denotes the singular values and  $\epsilon$  is a small threshold, then the rank is increased by one (i.e.,  $K = K + 1$ ) and the Eq. (12) is updated too. To update the Eq. (11), a partial  $\mathrm{SVD}(\mathbf{E} - \mathbf{S}^{(t)})$  is estimated keeping  $K$  components. To update the Eq. (12), a shrinkage operator is used  $\mathcal{D}(\cdot)$ , where:

$$
\mathcal{D}(\mathbf{E} - \mathbf{L}^{(t + 1)},\lambda) = \mathrm{sign}(\mathbf{E} - \mathbf{L}^{(t + 1)})\mathrm{max}\{0,|\mathbf{E} - \mathbf{L}^{(t + 1)}| - \lambda \} \tag{13}
$$
