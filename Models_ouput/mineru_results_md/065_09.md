We see a velocity gradient along the filament in the $\mathrm { D C O ^ { + } }$ (3- 2）gas.Therefore,we measured the ongoing accretion material toward the protostar core in this gas, $\dot { M } _ { \mathrm { a c c } } = 9 . 7 \times$ $1 0 ^ { - 7 } \mathrm { M } _ { \odot } \mathrm { y r } ^ { - 1 }$ , where the accretion rate is expected to be accuratewithinafactor of2. The mean velocity gradient is roughly $5 . 1 \mathrm { k m } \mathrm { s } ^ { - 1 } \mathrm { p c } ^ { - 1 }$ measured in $\mathrm { D C O ^ { + } }$ (3-2）around the protostar core,which is linked to the rotation of the core.This velocity gradient at the position of the protostar is in the east-west direction, oriented approximately perpendicular to the bipolar outflow previously found.   
Linewidths of $\mathrm { D C O ^ { + } }$ (3-2) increase toward the position of the protostar, probably due to protostellar feedback.   
We observed a velocity shift between neutral and ionized species.A higher velocity is always present in the $\mathrm { C } ^ { 1 8 } \mathrm { O }$ (2- 1) data compared to the $\mathrm { D C O ^ { + } }$ (3-2) data. The mean velocity difference, $\hat { V } _ { \mathrm { l s r } } ( { \bf C } ^ { 1 8 } 0 ) - V _ { \mathrm { l s r } } ( \mathrm { D C O ^ { + } } ) .$ is equal to $0 . 1 3 \mathrm { ~ k m s ^ { - 1 } }$ across the full filament.This is consistent witha model of collision between filaments that is still ongoing. The velocity shift between the $\mathrm { C } ^ { 1 8 } \mathrm { O }$ (2-1) and $\mathrm { D C O ^ { + } }$ (3-2) illustrates the relative motion of the dense gas,traced by $\mathrm { D C O ^ { + } }$ (3-2), and the surrounding less dense envelope,traced by $\mathrm { C } ^ { 1 8 } \mathrm { O }$ (2- 1).

Further observational investigations are needed to determine in more detail the connections within the kinematics and magnetic field in this source.
