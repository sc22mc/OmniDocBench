using that $w _ { \psi } = ( \cos \psi ) ^ { n - 2 } \leq 1$ .Then one concludes,that for $a > n$ large enough :

$$
\lambda W _ { n - 2 } < \left( 1 - \frac { 1 } { a \sqrt { n } } \right) \sqrt { \frac { \pi } { 2 n } } < \left( 1 - \frac { 1 } { a \sqrt { n } } \right) W _ { n - 2 }
$$

so that $\begin{array} { r } { \mathrm { I s o p } ( T M ) = \lambda + \frac { 1 } { a n W _ { n } } < \lambda + \frac { 1 } { a \sqrt { n } } < 1 = \mathrm { I s o p } ( F ) } \end{array}$ , as desired.
